# IT Help Desk Tracking System (Excel)

## Overview

This project is an Excel-based IT Help Desk Tracking System designed to simulate common IT support operations. It tracks employees, equipment, software assets, and support tickets while providing an automated dashboard to summarize key metrics.

## Features

- Employee record management
- Equipment inventory tracking
- Software license tracking
- IT support ticket management
- Automated dashboard
- Charts and visual reports
- Data validation for consistent data entry
- Organized worksheets for efficient workflow management

## Dashboard Metrics

- Total Employees
- Total Assets
- Open Tickets
- In Progress Tickets
- Closed Tickets

## Skills Demonstrated

- Microsoft Excel
- Dashboard Design
- Data Validation
- Data Analysis
- Reporting
- Inventory Management
- IT Asset Tracking
- Help Desk Operations

## Tools Used

- Microsoft Excel
- Excel Formulas
- Charts
- Conditional Formatting
- Data Validation

## Project Purpose

This project was created to simulate real-world IT help desk processes and demonstrate practical spreadsheet, reporting, and IT support skills relevant to entry-level IT Support and Help Desk roles.
